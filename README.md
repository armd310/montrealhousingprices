# montrealhousingprices
R map plot of median Montreal family home prices.<br>
Inspiration from: https://r-graph-gallery.com/web-map-choropleth-quantile.html <br>
Data taken from Centris 2025 Q4 housing prices for Montreal.<br>
Montreal map geojson downloaded from: https://donnees.montreal.ca/dataset/limites-administratives-agglomeration
