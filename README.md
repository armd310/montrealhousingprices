# Montreal Housing Prices Visualization

An R-based geospatial analysis and visualization of median family home prices across Montreal neighborhoods.

## Overview

This project analyzes and visualizes housing price data across different areas of Montreal using R and spatial mapping techniques. The visualization helps identify price trends and geographic patterns in the Montreal real estate market.

## Technologies Used

- **R** - Data analysis and visualization
- **ggplot2** - Data visualization (if you used it - adjust if different)
- **sf/sp packages** - Geospatial data handling (adjust based on what you actually used)

## Key Features

- Median home price analysis by neighborhood
- Static map visualization showing price distributions
- Geographic analysis of housing market trends
